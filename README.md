# FU-Bookkeeping

A simple HTML-based bookkeeping application for tracking income and expenses.

## Features

- ✅ Add income and expense transactions
- ✅ View transaction history sorted by date
- ✅ Track current balance with income/expense breakdown
- ✅ Persistent storage using browser's localStorage
- ✅ Clean and responsive design
- ✅ Delete individual transactions
- ✅ Clear all transactions

## How to Use

1. **Open the Application**
   - Simply open `index.html` in your web browser
   - No server or installation required!

2. **Add a Transaction**
   - Enter a description (e.g., "Salary", "Groceries")
   - Enter the amount
   - Select type (Income or Expense)
   - Choose the date
   - Click "Add Transaction"

3. **View Your Balance**
   - The balance card at the top shows your current balance
   - Income and expense totals are displayed separately

4. **Manage Transactions**
   - View all transactions in the history section
   - Delete individual transactions using the "Delete" button
   - Clear all transactions using the "Clear All" button

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - JavaScript functionality and data management

## Browser Compatibility

Works in all modern browsers that support:
- HTML5
- CSS3
- JavaScript ES6
- localStorage API

## Data Storage

All transaction data is stored locally in your browser using localStorage. This means:
- Your data persists between sessions
- Your data is private and never leaves your device
- Clearing browser data will remove all transactions
